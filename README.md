## Лабораторная работа "Обработка текстов с помощью нейронных сетей "

## Структура репозитория

- `lab4Lite.ipynb` –  ноутбук для задания Lite: подробная работа со словарем и формированием гиперпараметров нейронной сети. Создайте 9 нейросетей с различными гиперпараметрами для обработки текста
- `lab4Pro.ipynb` – ноутбук для задания Pro: обучение модели для распознавания позитивных и негативных отзывов людей по автомобилю Tesla
- `lab4Ultra_Pro.ipynb` – ноутбук для задания Ultra Pro:  обучение модели для распознавания текстов 20 русских писателей 
- `Pytorch_lab4.ipynb` – ноутбук для задания Lite:подробная работа со словарем и формированием гиперпараметров нейронной сети. Создайте 9 нейросетей с различными гиперпараметрами для обработки текста.Выполнено с использованием Pytorch

## Требования

Для запуска ноутбуков рекомендуется использовать **Google Colab**. Все необходимые библиотеки предустановлены в Colab.
